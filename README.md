# Jevon Harry Irmanto | Computer Engineering Portfolio 

Welcome to the repository for my professional portfolio website. This project serves as a centralized hub to showcase my academic milestones, engineering designs, and technical competencies in computer and electrical systems.

The live site can be accessed at: **[jevonharry.com](https://jevonharry.com)** *(Temporary mirror: [jevonharry.github.io](https://jevonharry.github.io))*

---

## 🛠️ Tech Stack & Architecture

- **Frontend:** Semantic HTML5, Custom CSS3 Grid/Flexbox layouts, Vanilla JavaScript.
- **Hosting:** GitHub Pages (Static hosting pipeline).
- **Domain & DNS Management:** Namecheap Custom Domain Mapping with Cloudflare/GitHub automated SSL (HTTPS) enforcement.
- **Development Workflow:** Clean, modular structure using a decoupled `assets/` directory (`/css`, `/js`, `/images`) to allow for simple future data expansion.

## ⚙️ Core Technical Highlights Showcased

This portfolio highlights key engineering competencies, including:
- **Autonomous Systems:** Implementation details of an autonomous line-tracking robot featuring optimized H-bridge motor logic built in C++.
- **Hardware & Semiconductors:** Practical experience with lithography, etching, and advanced pattern transfer via semiconductor nanofabrication training at UCLA.
- **Digital Manufacturing:** Product layout modeling and slicing optimized for Bambu Lab hardware, showcased at the PolyX Exhibition.
- **Circuit Design & Simulation:** Analytical validation and modeling of circuit elements (MOSFETs, BJTs) utilizing LTspice.
- **Building Systems Design:** Systems integration utilizing Revit MEP software alongside National Electrical Code (NEC) standards.
